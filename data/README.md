# Data

Place the M3 Challenge data file here:

- `M3-Challenge-Problem-Data-2026.xlsx` — official M3 Competition spreadsheet
  - Sheet `"Expenditures (U.K.)"` — UK household expenditure by age group and income quintile
  - Sheet `"Expenditures (U.S.)"` — US consumer expenditure survey data

The file is not included in this repo. Download it from the official M3 Challenge portal.
All hardcoded values in `src/data.py` were transcribed from this file.
